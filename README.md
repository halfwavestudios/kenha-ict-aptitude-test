# KeNHA ICT Officer II Aptitude Test Web App

Interactive **60-question, 60-minute** online aptitude test designed for the **ICT Officer II (Grade 8)** position at Kenya National Highways Authority (**KeNHA**).

Features:
- One question per page
- 60-minute countdown timer (auto-submits when time is up)
- Session-based answer persistence
- Detailed results with score, percentage, and correct answers shown
- Covers: General Reasoning, Basic ICT, Networking, Databases, Web Development & PHP

## Tech Stack
- PHP 7.4+
- HTML5 + CSS3
- JavaScript (for client-side timer)
- No database required (questions in plain PHP array)
